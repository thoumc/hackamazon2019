## Setup

1. Fork & Clone
2. Run `bundle install` to install dependencies
3. Run `bin/rake db:reset` to create, load and seed db
4. Create .env file based on .env.example
5. Run `bin/rails s` to start the server

## Dependencies:
- Rails 5.2
- SQLite3
- Bcrypt
